주짓수 스코어판 
***

* 개발 환경 : HTML, CSS, JavaScript, JQuery

***
Web 화면



***
Moblie 화면

* 기능 
  * 주짓수 배경으로한 스코어판
  * 이름 및 시간 지정 (기본값: Player1,2 시간 : 5분)
  * START를 누르면 시작
  * 숫자를 누르면 점수를 추가 가능
  * CANCLE를 누르면 이전에 획득한 점수 취소 
  * Log를 누르면 이전까지 얻은 점수를 한번에 확인 가능
  * PAUSE를 누르면 시간 및 점수 획득 불가능
  * END를 누르면 게임 종료 및 승자 판정 가능 
  * 판정 후 CANCLE -> RESET으로 바뀌고 점수판 고정
